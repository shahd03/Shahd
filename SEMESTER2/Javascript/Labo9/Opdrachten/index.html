  • Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?

    - Nee, als het tabblad wordt gesloten, worden ook alle variabelen die in dat tabblad zijn opgeslagen, gewist.

  • Gebruiken twee tabbladen die dezelfde pagina tonen, dezelfde data?

    - Nee, elke tabbladsessie heeft zijn eigen instantie van de webpagina en zijn eigen JavaScript-uitvoeringscontext.

Opdracht demo 2 session storage:

  • Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?

    - Ja, gegevens die zijn opgeslagen in de session storage blijven behouden, zelfs als je het tabblad sluit en de pagina opnieuw opent.
    Session storage wordt gewist zodra het tabblad of de browser wordt gesloten.

  • Blijft de data behouden als je de pagina herlaadt?

    - Ja, de data blijft behouden bij het herladen van de pagina.

  • Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?

    - Nee, session storage is beperkt tot het huidige tabblad.
    Gegevens in session storage worden niet gedeeld tussen verschillende tabbladen of vensters.

  • Denk je dat een andere pagina (andere file of url) aan de data geraakt als je ze in hetzelfde tabblad opent?

    - Nee, session storage is beperkt tot de huidige domeincontext en wordt niet gedeeld tussen verschillende pagina's of bestanden, zelfs niet binnen hetzelfde tabblad.
    Elke pagina heeft zijn eigen aparte session storage.

Opdracht demo 3 local storage:

  • Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?

    - Ja, de data blijft behouden in de local storage, zelfs als je het tabblad sluit en de pagina opnieuw opent.

  • Blijft de data behouden als je de pagina herlaadt?

    - Ja, de data blijft behouden bij het herladen van de pagina.

  • Blijft de data behouden als je de browser afsluit, opnieuw start en de pagina weer opent?

    - Ja, de data blijft behouden in de local storage, zelfs als je de browser afsluit en opnieuw start.

  • Blijft de data behouden als je de pagina in een andere browser opent?

    - Nee, local storage is specifiek voor elke browser. De data is niet toegankelijk in een andere browser.

  • Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?

    - Ja, local storage is beschikbaar voor alle tabbladen die dezelfde pagina openen. Ze delen dezelfde local storage.

  • Denk je dat een andere pagina (andere file of url) aan de data geraakt?

    - Nee, local storage is beperkt tot de domeincontext. Andere pagina's kunnen de data niet rechtstreeks aanraken.
